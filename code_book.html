Codebook for Means Dataset

Introduction

This codebook describes the variables in the means dataset, which is derived from the Human Activity Recognition Using Smartphones dataset. The dataset contains the mean values of various measurements grouped by subject and activity. Units: the measurements have been standardized so they don’t have units

Variables

subject

  • Description: Identifier of the subject who carried out the experiment.
  • Type: Integer
  • Range: 1 to 30

activity

  • Description: Activity performed by the subject.
  • Type: Factor
  • Levels: LAYING, SITTING, STANDING, WALKING, WALKING_DOWNSTAIRS, WALKING_UPSTAIRS
